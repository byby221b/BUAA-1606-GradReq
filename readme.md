### :star:我达到最低学分要求了吗？

* 总学分：**134**
* 数学与自然科学、工程基础、语言类课程：**33.5**
* 思政、军理、体育、博雅类课程：**20**
* 核心通识类课程：**12**
* 专业必修课程：**50.5**
* 专业选修课：**18**

### :electric_plug:我达到核心通识要求了吗？

* 我的核心通识是否选够**六门**？
* 我的每门核心通识是否都**不少于 2 学分**？
* 我选修**航概A、经管**了吗？
* 我的核心通识里面有**法律类课程**吗？

### :earth_africa:我达到一般专业课选课要求了吗？

* 我的某一方向的选修课程是否不少于五门？

  >  （培养方案表述为：**建议**）

* 我是否选修了**其它**专业的专业必修课或者专业选修课？（**非讲座类课程，2学分以上**）

* 我是否选修过**全英文课程**？（**2学分以上**）

### :balloon:我的*TD*刷够了吗？

* 每个学期都有刷卡记录，且总数超过288
